# 문제 해결

개발 과정에서 발생할 수 있는 일반적인 문제들과 해결 방법을 알아보겠습니다.

## 일반적인 문제

### 1. 빌드 오류

**문제**: `npm run build` 실행 시 오류 발생

**해결 방법**:
```bash
# 캐시 정리
npm cache clean --force

# node_modules 삭제 후 재설치
rm -rf node_modules package-lock.json
npm install

# 빌드 재시도
npm run build
```

### 2. 의존성 충돌

**문제**: 패키지 버전 충돌로 인한 설치 실패

**해결 방법**:
```bash
# 의존성 트리 확인
npm ls

# 특정 패키지 버전 확인
npm list package-name

# 강제 설치
npm install --force
```

### 3. 포트 충돌

**문제**: 포트가 이미 사용 중이라는 오류

**해결 방법**:
```bash
# 포트 사용 중인 프로세스 확인
lsof -i :3000

# 프로세스 종료
kill -9 PID

# 다른 포트 사용
PORT=3001 npm start
```

## API 관련 문제

### 1. CORS 오류

**문제**: 브라우저에서 CORS 정책 위반 오류

**해결 방법**:
```javascript
// Express.js CORS 설정
const cors = require('cors');

app.use(cors({
  origin: ['http://localhost:3000', 'https://yourdomain.com'],
  credentials: true
}));
```

### 2. 인증 토큰 만료

**문제**: API 요청 시 401 Unauthorized 오류

**해결 방법**:
```javascript
// 토큰 자동 갱신
const refreshToken = async () => {
  try {
    const response = await fetch('/api/refresh-token', {
      method: 'POST',
      credentials: 'include'
    });
    
    if (response.ok) {
      const data = await response.json();
      localStorage.setItem('accessToken', data.accessToken);
    }
  } catch (error) {
    // 로그인 페이지로 리다이렉트
    window.location.href = '/login';
  }
};
```

## 성능 문제

### 1. 메모리 누수

**문제**: 애플리케이션 메모리 사용량 증가

**해결 방법**:
```javascript
// 이벤트 리스너 정리
useEffect(() => {
  const handleResize = () => {
    // 리사이즈 처리
  };
  
  window.addEventListener('resize', handleResize);
  
  return () => {
    window.removeEventListener('resize', handleResize);
  };
}, []);
```

### 2. 번들 크기 최적화

**문제**: JavaScript 번들 크기가 너무 큼

**해결 방법**:
```javascript
// 동적 임포트 사용
const LazyComponent = React.lazy(() => import('./LazyComponent'));

// 코드 스플리팅
const routes = [
  {
    path: '/dashboard',
    component: React.lazy(() => import('./Dashboard'))
  }
];
```

### 2. 네트워크 모니터링

```javascript
// API 요청 모니터링
const originalFetch = window.fetch;
window.fetch = function(...args) {
  console.log('API 요청:', args[0]);
  return originalFetch.apply(this, args)
    .then(response => {
      console.log('API 응답:', response);
      return response;
    });
};
```
